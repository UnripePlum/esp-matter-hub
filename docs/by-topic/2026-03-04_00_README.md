## 현재 기준 상태 (2026-03-10)
- Matter 브리지 + 로컬 웹 운영 흐름 동작
- mDNS/IP 기반 접근 지원
- IR 학습/커밋/목록 조회 동작
- endpoint 8개 고정 + 등록 기기 최대 16 + 수동 할당 모델 적용
- 슬롯 바인딩: `on_signal_id`, `off_signal_id`, `level_up_signal_id`, `level_down_signal_id`
- Level 제어 방향 기반 송신(밝기 0 -> off)
- 상태 LED 연동 및 `/api/health` 상태 노출
- NVS export API 제공: `GET /api/export/nvs`
- NodeLabel 동기화: assign/unassign/rename 시 endpoint 표시명 동기 반영
- 운영/개편 이원화 계획: 평면 8슬롯 운영 라인 유지 + Aggregator PoC/안정화 트랙 분리
